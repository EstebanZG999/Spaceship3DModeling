## Getting Started

### Prerequisites
- Rust (latest version)

### Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/EstebanZG999/Spaceship3DModeling.git
   ```
2. Navigate to the project directory:
   ```sh
   cd Spaceship3DModeling
   ```
3. Build the project:
   ```sh
   cargo build
   ```

## Usage
To run the application, use the following command:
```sh
cargo run
```
